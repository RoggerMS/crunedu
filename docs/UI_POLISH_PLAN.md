# UI_POLISH_PLAN

## Objetivo

Pulir la experiencia principal del App Shell y Feed para que el flujo diario sea compacto, claro y escalable en desktop y móvil.

## Alcance implementado en este paso

1. Inicio con botones compactos y accionables.
2. Sección reutilizable separada en `/app/reutilizable`.
3. Acceso base a notificaciones en `/app/notificaciones`.
4. Iconos rápidos de notificaciones y perfil en header.
5. CTA de crear comunidad en módulo Comunidades.
