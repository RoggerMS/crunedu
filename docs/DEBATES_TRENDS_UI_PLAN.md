# Debates Trends UI Plan

## Objetivo
Adaptar la página de Debates a una navegación escalable por filtros y ventanas de tendencia, manteniendo el MVP simple.

## Alcance MVP inmediato
1. Reemplazar el selector principal por dos ámbitos:
   - Académico
   - No académico
2. Mostrar canales en listas desplegables por sección:
   - Cursos generales
   - Cursos de especialidad
   - Vida universitaria y otros
3. Agregar pestañas de tendencia:
   - Debates del día
   - Debates de la semana
   - Debates del mes
4. Mantener flujo existente de crear debate y responder.

## Pendiente siguiente iteración
- Reacción apoyo/no apoyo (upvote/downvote).
- Imagen opcional en creación de debate.
- Ranking persistente por ventana temporal.
- Modo histórico/reutilizable (anual/all-time) en `/app/reutilizable`.
