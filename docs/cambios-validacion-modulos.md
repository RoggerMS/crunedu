# Cambios de navegación y módulos (MVP)

Fecha: 2026-05-01

## Qué se ajustó

- Se agregó la pestaña **Debates** en la navegación principal lateral del app shell.
- Se reforzó la sección **Momentos** para que no sea solo informativa: ahora incluye una lista de momentos relevantes y una interacción de votos con botón "Impulsar momento".

## Objetivo

Alinear la experiencia por módulos para que cada pestaña tenga una intención propia y participación directa del estudiante.

## Alcance

- No se modificó la lógica de autenticación.
- No se modificaron endpoints del API.
- No se alteró la funcionalidad existente de comunidades.
